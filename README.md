# Honkai Star Rail: Build Calculator

Honkai Star Rail: Build Calculator é um site na web que fornece uma previsão de _builds_ para personagens do jogo Honkai Star Rail.

## Sobre o projeto

Esse projeto tem como foco a criação de um site voltado para a comunidade do jogo Honkai Star Rail, visando diminuir o retrabalho de criação de _builds_ dentro do jogo e também, para que forneça uma atualização o máximo automatizada possível, para acompanhar os patchs do jogo. O propósito reside na dificuldade dos jogadores em saber qual o melhor artefato, muitos sites apresentam textos manuais e até mesmo errados sobreq como construir o seu personagem com a infinidade de artefatos. 

Além disso, ele irá contar com coisas padrões de outros sites, como análise de _skills_ e _level up_ para dar uma dimensão melhor para o jogador.

## Instalação
> [!IMPORTANT]
> Se você já sabe como instalar as depedências, por favor, pule para o próximo tópico.

<details>
<summary>
    Guia de Instalação
</summary
  
Siga os passos abaixo para rodar a aplicação localmente em sua máquina. 

### 1. Clonando o repositório

```bash
https://github.com/CIGBORS/starrail-build-calculator.git
```

### 2. Instalando as dependência

#### 2.1. Frontend
Acesse a pasta correspondente ao frontend.

```bash
cd frontend
```

Instale todas as dependência do frontend.

```bash
npm install
```

#### 2.2. Backend

Acesse a pasta correspondente ao backend após a instalação do frontend.

```
cd ..
cd backend
```

Instale todas as dependências do backend.

```
npm install
```

### 3. Configuração das variáveis de ambiente (env)

Crie um arquivo `.env` na raiz do projeto.

Exemplo de configuração do arquivo:

```env
# POSTGRESQL
POSTGRESQL_HOST=localhost
POSTGRESQL_PORT=5432
POSTGRESQL_USER=order_api
POSTGRESQL_PASSWORD=your_password
POSTGRESQL_DATABASE=orders

# BACKEND
BACKEND_PORT=3000
```

### 4. Ligue o Servidor

```bash
npm start
```

A APi estará disponível em:

```url
http://localhost:3000
```
